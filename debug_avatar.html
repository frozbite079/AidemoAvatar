🚨 Common Issues & Solutions

  • Avatar area is white/empty: You need to click "Start Avatar" first
  • "Start Avatar" button missing: Open the assistant popup first
  • Connection fails: Check Azure Speech credentials
  • Browser blocks camera/mic: Allow permissions when prompted

Step 4: Manual Avatar Test

If all above tests pass, the issue is likely that you need to:

  1. Go to http://localhost:8000
  2. Click "Ask Musaed" to open the popup
  3. Look for "Avatar Settings" section
  4. Click "Start Avatar" button
  5. Wait for "Avatar connected" status
