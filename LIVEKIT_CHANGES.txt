================================================================================
PERFORMANCE CHARACTERISTICS
================================================================================

UI Performance:
  - Page load: ~500ms
  - Join room: ~1-2 seconds
  - WebRTC connection: ~2-5 seconds
  - Audio latency: <100ms (LiveKit Cloud)
  - Video latency: <150ms (LiveKit Cloud)

Memory Usage:
  - Frontend: ~50MB with video feeds
  - Backend: Minimal (just token generation)
  - Per participant: ~20MB video buffer

Network Bandwidth:
  - Audio: ~32-64 kbps
  - Video (HD): ~1-5 Mbps
  - Data messages: Minimal

Scalability:
  - Per room: Unlimited users (limited by bandwidth)
  - Per LiveKit server: 1000+ concurrent users
  - Multiple rooms: Full isolation
